# アプリ名：家計簿戦略

## サイト概要
家計の収支を簡単に管理するアプリ。
支出は「消費」「浪費」「投資」の3つに分類し、資金使徒をグラフで可視化することで、お金の有意義な使い方を促す。(自戒の念を込めて)
