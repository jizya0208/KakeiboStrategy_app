# アプリ名：家計簿戦略

## サイト概要
家計の収支を簡単に管理するアプリ。支出項目においては、
資金使徒を「消費」「浪費」「投資」の3つに分類し、グラフでの可視化によって、
より有意義なお金の使い方を促します。(自戒の念を込めて…)
